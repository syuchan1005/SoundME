# SoundMeとは
 WebでiTunesみたいの欲しかっただけ
 
# どんな感じなの？
   iTunesライクな感じのデザインです  
   [Demo](http://test.magitech.xyz)  <- ちょっと重たいかも
 
# 使い方
 `npm install`して`npm start`してください
 
 logが見たい場合には`DEBUG=soundme`をつけてください
 
 初期ユーザーは
 ```text
User: user
Pass: pass
```
  になっています
